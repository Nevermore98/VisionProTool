# VisionPro Tool

VisionPro Tool（康耐视 VisionPro 调试工具），使用 WPF + [WPF-UI](https://github.com/lepoco/wpfui) + [MVVM Toolkit](https://github.com/CommunityToolkit/dotnet) + IOC 构建

便于调试开发 ToolBlock，而无需打开 VisionPro QuickBuild 应用
